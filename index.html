Yggdrasil Proxy

О шлюзе

Этот прокси-сервер позволяет получать доступ к сайтам внутри независимой сети Yggdrasil из обычного интернета без необходимости устанавливать клиент сети.

Шлюз полностью изолирован: он строго проверяет входящие запросы и маршрутизирует трафик только внутри доверенной подсети 200::/7, блокируя любые попытки использования сервера в качестве публичной веб-прокси.
